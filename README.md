# AI Image Generator 

A full-stack AI Image Generator web application built using **React (Frontend)** and **Node.js + Express (Backend)**.  
The application allows users to enter a text prompt and generate AI images in real time using a free public image generation API.

---

##  Features

-  Generate AI images from text prompts  
-  Real-time image rendering  
-  Modern UI with gradient background and sample image sidebars  
-  Secure setup (no API keys exposed in frontend)  
-  REST API architecture  

---

## Tech Stack

### Frontend
- React.js
- HTML5, CSS3
- JavaScript (ES6)
- Fetch API

### Backend
- Node.js
- Express.js
- CORS
- Free AI Image API (Pollinations)
